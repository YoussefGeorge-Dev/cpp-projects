## 🚀 How to Run

### Visual Studio Code

1. Install the [C/C++ Extension Pack](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools-extension-pack)
2. Open any `.cpp` file
3. Click the **▷ Run** button

### Command Line

```bash
g++ Lab04.cpp -o output && ./output
```
